Latitude vs X

The Purpose of this project is to analyze how weather changes as you get closer to the equator. To accomplish this analyzsis we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used matplotlib to plot various aspects of the weather vs latitude. Factors we looked at included: temperature, cloudiness, humidity, and windspeed. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and distriptions of any trends and correlations witnessed.
